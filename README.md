# Bookstore API

Esta é uma API para gerenciar um sistema de livraria, onde é possível registrar usuários, realizar login e gerenciar livros. Foi construída utilizando **Node.js**, **Express**, **TypeScript**, e **PostgreSQL** como banco de dados.

## Tecnologias Usadas

- **Node.js** - Ambiente de execução JavaScript no lado do servidor
- **Express** - Framework para construção de APIs RESTful
- **TypeScript** - Superset de JavaScript com tipagem estática
- **PostgreSQL** - Banco de dados relacional
- **pg** - Biblioteca para interação com o PostgreSQL
- **dotenv** - Carregamento de variáveis de ambiente a partir de um arquivo `.env`
- **Jest** - Framework de testes
- **ts-jest** - Integração do Jest com TypeScript

## Pré-requisitos

Antes de rodar a aplicação, certifique-se de ter as seguintes dependências instaladas:

- [Node.js](https://nodejs.org/) (versão >= 14.x)
- [PostgreSQL](https://www.postgresql.org/) (ou um serviço de banco de dados como Supabase)
- [TypeScript](https://www.typescriptlang.org/) (para desenvolvimento)
